# vibecoding-optimization

English | [中文](./README_CN.md)

This is a skill designed for programming tasks. It improves the vibecoding experience and enhances code reliability, readability, and maintainability.

## Installation

```bash
npx skills elysias123/vibecoding-optimization
```

## Usage

- Automatically loads during programming development and related tasks
- Use `/vibecoding <on/off>` or `/vibecoding-optimization <on/off>` to control it
